# 🍅 番茄钟 (Pomodoro Timer)

一个带图形界面的番茄钟桌面应用，帮助提高专注力。暖色调 + 圆形进度条设计。

## 功能

- ⏱ 自定义工作时间、短休息、长休息
- 🔄 自定义每轮番茄数
- 🎨 暖色调配色，不同阶段不同颜色
- 🍅 圆形进度条实时显示倒计时
- ⏸ 随时暂停/继续
- 🧘 自动切换工作/短休/长休
- 🔊 不同事件有独立音效提示

## 使用方法

```bash
py pomodoro_gui.py
```

### 参数设置

在窗口中自定义：

| 参数 | 默认值 | 范围 |
|------|--------|------|
| ⏱ 工作时间 | 25 分钟 | 1–120 |
| ☕ 短休息 | 5 分钟 | 1–30 |
| 🧘 长休息 | 15 分钟 | 1–60 |
| 🔄 每轮番茄 | 4 个 | 1–10 |

设置完成后点击 **✅ 应用**，然后点击 **▶ 开始** 启动计时。

### 音效

| 事件 | 说明 |
|------|------|
| ▶ 开始 | 启动计时时播放 |
| ⏰ 工作结束 | 工作时段结束时播放 |
| ☕ 短休结束 | 短休息结束时播放 |
| 🧘 长休结束 | 长休息结束时播放 |

音效文件放在 `sounds/` 目录下，可自行替换为喜欢的 `.wav` 文件。

## 运行环境

- Python 3.x（自带 tkinter 库，无需额外安装）
- 仅支持 Windows（使用 winsound）
