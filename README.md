# 🍅 番茄钟 (Pomodoro Timer)

一个带图形界面的番茄钟桌面应用，帮助提高专注力。暖色调 + 圆形进度条设计。

## 功能

- ⏱ 自定义工作时间、短休息、长休息
- 🔄 自定义每轮番茄数
- 🎨 暖色调配色，不同阶段不同颜色
- 🍅 圆形进度条实时显示倒计时
- ⏸ 随时暂停/继续
- 🧘 自动切换工作/短休/长休

## 使用方法

```bash
py pomodoro_gui.py
```

### 参数设置

在窗口中自定义：
| 参数 | 默认值 | 范围 |
|------|--------|------|
| 工作时间 | 25 分钟 | 1–120 |
| 短休息 | 5 分钟 | 1–30 |
| 长休息 | 15 分钟 | 1–60 |
| 每轮番茄 | 4 个 | 1–10 |

设置完成后点击 **✅ 应用**，然后点击 **▶ 开始** 启动计时。
